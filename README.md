# 🌍 Ambiente, Educazione Civica & Intelligenza Artificiale
## Capolavoro Scolastico - Classe V Superiore

---

## 📋 Descrizione del Progetto

Questo sito web innovativo e interattivo è stato sviluppato come capolavoro scolastico per affrontare tre tematiche fondamentali:

1. **🌱 Educazione Civica** - I diritti e doveri del cittadino verso l'ambiente
2. **🌍 Questioni Ambientali** - La crisi climatica, l'inquinamento e la biodiversità
3. **🤖 Intelligenza Artificiale** - Come la tecnologia può salvare il nostro pianeta

---

## 🎯 Obiettivi Pedagogici

- **Consapevolezza civica**: Comprendere il ruolo del cittadino nella tutela ambientale
- **Pensiero critico**: Analizzare problemi globali complessi
- **Innovazione tecnologica**: Esplorare soluzioni sostenibili attraverso l'IA
- **Responsabilità personale**: Identificare azioni concrete quotidiane

## 🚀 Come Utilizzare il Sito

### Opzione 1: Locale (Offline)
1. Scarica i tre file (index.html, styles.css, script.js)
2. Apri `index.html` nel browser
3. Naviga tra le sezioni

### Opzione 2: Online (Con Server)
```bash
# Se hai Python 3
python -m http.server 8000

# Se hai Node.js
npx http-server

# Se hai Live Server in VS Code
Apri con Live Server
```

### Opzione 3: GitHub Pages
1. Crea un repository GitHub
2. Carica i file
3. Abilita GitHub Pages nelle settings
4. Accedi a `https://tuonome.github.io/nomeprogetto`
